# Dynamic Infinite Maze

A pseudo-3D game written in Python, utilizing the classic **Raycasting** technique — a rendering technology.

## 🛠️ Features
* **Modular Architecture:** The codebase is split into clean, logical modules for easy scalability.
* **Infinite Generation:** Dynamic maze generation ensures a unique gameplay experience every run.
* **Retro Atmosphere:** Real-time raycasting calculations without relying on heavy 3D engines.

## 📁 Project Structure
* `doom.py` — The main entry point to launch the game.
* `config.py` — Game settings, graphics resolution, and controls.
* `engine.py` — The mathematical core handling raycasting and ray calculations.
* `interface.py` — UI rendering and screen drawing logic.
* `player.py` — Player movement physics and collision detection.

## 🚀 How to Run
1. Ensure you have Python 3.x installed.
2. Download all repository files into a single directory.
3. Run the main script:
   ```bash
   python doom.py
   ```
